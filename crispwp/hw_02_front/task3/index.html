Задание:

Найдите количество Килобайт (KB) и Байт (B) в данном сообщении.

В 1 Килобайте -- 1024 Байта;
В 1 Байте -- 8 бит;

Решение:

При решении данного задания воспользуемся формулой Хартли:
N = 2 i
N - мощность алфавита
i - информационный вес одного символа ( в битах )

Мощность русского алфавита со всеми дополнительными символами - 54 ( N = 54 )

Находим количество информации, которое несет один знак в сообщении ( i ):
54 = 2 i = 2 7.755
i = 5.755 бит

Выведем формулу для вычисления информационного объёма сообщения:
I = K * i
I - информационный объём сообщения
K - количество символов в сообщении

Количество символов ( I ) - 112

Находим информационный объём сообщения ( I ):
I = 112 * 5.755 = 644.56 бит

Ответ:

Объём сообщения

644.56 бит

80.57 байт

0.079 килобайт
