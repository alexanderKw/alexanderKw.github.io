Задание:

Исследуйте код, найдите выражения в нём. Расставьте скобки так, чтобы код не изменил своего поведения (работал так же как и сейчас). Объясните как и в каком порядке вычисляются выражения и подвыражения.

var a = 5;
var b, c;
b = a * 5;
b = c = b / 2;

Решение:

var a = 5; - переменной a присваивается значение 5.

var b, c; - объявление переменных b и c

b = a * 5; - здесь код выполняется справа налево. Сначала выполняется выражение, которое находится справа от знака "=", а после, результат этого выражения присваивается переменной b

b = c = b / 2; код выполняется справа налево. Сначала выполняется выражение, которое находится справа от второго знака "=", результат этого выражения присваивается переменной c. После, значение переменной c присваивается переменной b.
Этот код можно записать так: b = c = (a * 5) / 2; - правому операнду второго знака "=", заменить переменную b выражением (a * 5)
